Detailed Description

/brief Abstract iterator class.

After tuples are inserted into a relation, they will be stored in consecutive memory space. Intially, the iterator_base of a relation will point to the first tuple. It can be moved to point to the next tuple until the end. The tuple iterator_base is pointing to can be accessed. However, users can not use this to access tuples. Instead, they should use iterator which interacts with iterator_base.
